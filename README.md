
# Node.js playground

A Node.js E-Commerce API backend monolithic project.


## Features

- User authentication and management
- Inventory
- Orders


## Tech Stack

**Server:** Node.js, Express, MongoDB.


## Installation

Install using npm

```bash
cd to the root of project
npm run start
  
```
